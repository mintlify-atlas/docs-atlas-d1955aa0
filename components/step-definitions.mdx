title: 'Step definitions'
description: 'StepDefinition maps every Gherkin step to Java code, wiring together the request spec, test data builder, and response assertions.'

`StepDefinition` extends `Utils` and contains the six methods that implement every Gherkin step used in the feature files. It also holds the static `place_id` field shared across scenarios.

## The `place_id` static field

```java
public static String place_id;
```

`place_id` is declared `static` so that its value persists across separate scenario instances. Cucumber creates a new `StepDefinition` object for each scenario, so instance fields reset between scenarios. Making `place_id` static allows the Delete scenario (and the `@Before` hook) to read the ID captured during a previous Add Place scenario.

## Step definitions

<AccordionGroup>
  <Accordion title="@Given — add Place Payload with {string} {string} {string}">
    **Gherkin example:**
    ```gherkin
    Given add Place Payload with "Mahmoud" "Arabic" "Africa"
    ```

    **Implementation:**
    ```java
    @Given("add Place Payload with {string} {string} {string}")
    public void add_place_payload_with(String name, String language, String address) throws IOException {
        requestAdd = given().spec(requestSpecification())
            .body(data.addPlacePayload(name, language, address));
    }
    ```

    Calls `requestSpecification()` (inherited from `Utils`) to get the shared spec, then attaches the Add Place JSON body built by `TestDataBuild.addPlacePayload()`. The three Gherkin string parameters are passed directly to the payload builder as `name`, `language`, and `address`.
  </Accordion>

  <Accordion title="@Given — DeletePlace Payload">
    **Gherkin example:**
    ```gherkin
    Given DeletePlace Payload
    ```

    **Implementation:**
    ```java
    @Given("DeletePlace Payload")
    public void delete_place_payload() throws IOException {
        requestAdd = given().spec(requestSpecification())
            .body(data.deletePlacePayload(place_id));
    }
    ```

    Reads the static `place_id` field and passes it to `TestDataBuild.deletePlacePayload()`, which builds the raw JSON string `{"place_id": "<id>"}`. The step takes no Gherkin parameters because the ID is captured automatically during earlier steps.
  </Accordion>

  <Accordion title="@When — user calls {string} with {string} http request">
    **Gherkin example:**
    ```gherkin
    When user calls "addPlaceAPI" with "POST" http request
    ```

    **Implementation:**
    ```java
    @When("user calls {string} with {string} http request")
    public void user_calls_with_http_request(String resource, String method) {
        APIResources resourceAPI = APIResources.valueOf(resource);
        resSpec = new ResponseSpecBuilder()
            .expectStatusCode(200)
            .expectContentType(ContentType.JSON);
        
        if (method.equalsIgnoreCase("post")) {
            response = requestAdd.when().post(resourceAPI.getResource());
        } else if (method.equalsIgnoreCase("get")) {
            response = requestAdd.when().get(resourceAPI.getResource());
        }
    }
    ```

    Converts the `resource` string to an `APIResources` enum constant via `valueOf()`, then routes the request to either `post()` or `get()` based on the `method` parameter. The response is stored in the instance field `response` for use by subsequent `@Then` steps.
  </Accordion>

  <Accordion title="@Then — the API call got success with status code {int}">
    **Gherkin example:**
    ```gherkin
    Then the API call got success with status code 200
    ```

    **Implementation:**
    ```java
    @Then("the API call got success with status code {int}")
    public void the_api_call_got_success_with_status_code(int statusCode) {
        assertEquals(statusCode, response.getStatusCode());
    }
    ```

    Asserts that the HTTP status code in `response` matches the integer provided in the Gherkin step. Uses JUnit's `assertEquals`.
  </Accordion>

  <Accordion title="@Then — {string} in response body is {string}">
    **Gherkin example:**
    ```gherkin
    Then "status" in response body is "OK"
    ```

    **Implementation:**
    ```java
    @Then("{string} in response body is {string}")
    public void in_response_body_is(String actualKeyValue, String expectedKeyValue) {
        assertEquals(getJsonPath(response, actualKeyValue), expectedKeyValue);
    }
    ```

    Extracts the value at JSON key `actualKeyValue` from the response body using `getJsonPath()` (inherited from `Utils`) and asserts it equals `expectedKeyValue`.
  </Accordion>

  <Accordion title="@Then — verify place_Id created maps to {string} using {string}">
    **Gherkin example:**
    ```gherkin
    Then verify place_Id created maps to "Mahmoud" using "getPlaceAPI"
    ```

    **Implementation:**
    ```java
    @Then("verify place_Id created maps to {string} using {string}")
    public void verify_place_id_created_maps_to_using(String expectedName, String resource) throws IOException {
        place_id = getJsonPath(response, "place_id");
        requestAdd = given().spec(requestSpecification()).queryParam("place_id", place_id);
        user_calls_with_http_request(resource, "GET");
        String actualName = getJsonPath(response, "name");
        assertEquals(actualName, expectedName);
    }
    ```

    Captures `place_id` from the current response and stores it in the static field. Then makes a GET call to the specified resource with `place_id` as a query parameter, and asserts that the `name` in the response matches `expectedName`. This step both stores the ID for later use and verifies the Add Place operation succeeded.
  </Accordion>
</AccordionGroup>

## Full source code

```java StepDefinition.java
package stepDefinitions;

import io.cucumber.java.en.Given;
import io.cucumber.java.en.Then;
import io.cucumber.java.en.When;
import io.restassured.builder.ResponseSpecBuilder;
import io.restassured.http.ContentType;
import io.restassured.response.Response;
import io.restassured.specification.RequestSpecification;
import io.restassured.specification.ResponseSpecification;
import resources.APIResources;
import resources.TestDataBuild;
import resources.Utils;

import java.io.IOException;

import static io.restassured.RestAssured.given;
import static org.junit.Assert.assertEquals;

public class StepDefinition extends Utils {
    
    RequestSpecification requestAdd;
    Response response;
    ResponseSpecification resSpec;
    public static String place_id;
    TestDataBuild data = new TestDataBuild();
    
    @Given("add Place Payload with {string} {string} {string}")
    public void add_place_payload_with(String name, String language, String address) throws IOException {
        requestAdd = given().spec(requestSpecification())
            .body(data.addPlacePayload(name, language, address));
    }
    
    @Given("DeletePlace Payload")
    public void delete_place_payload() throws IOException {
        requestAdd = given().spec(requestSpecification())
            .body(data.deletePlacePayload(place_id));
    }
    
    @When("user calls {string} with {string} http request")
    public void user_calls_with_http_request(String resource, String method) {
        APIResources resourceAPI = APIResources.valueOf(resource);
        resSpec = new ResponseSpecBuilder().expectStatusCode(200).expectContentType(ContentType.JSON);
        
        if (method.equalsIgnoreCase("post")) {
            response = requestAdd.when().post(resourceAPI.getResource());
        } else if (method.equalsIgnoreCase("get")) {
            response = requestAdd.when().get(resourceAPI.getResource());
        }
    }
    
    @Then("the API call got success with status code {int}")
    public void the_api_call_got_success_with_status_code(int statusCode) {
        assertEquals(statusCode, response.getStatusCode());
    }
    
    @Then("{string} in response body is {string}")
    public void in_response_body_is(String actualKeyValue, String expectedKeyValue) {
        assertEquals(getJsonPath(response, actualKeyValue), expectedKeyValue);
    }
    
    @Then("verify place_Id created maps to {string} using {string}")
    public void verify_place_id_created_maps_to_using(String expectedName, String resource) throws IOException {
        place_id = getJsonPath(response, "place_id");
        requestAdd = given().spec(requestSpecification()).queryParam("place_id", place_id);
        user_calls_with_http_request(resource, "GET");
        String actualName = getJsonPath(response, "name");
        assertEquals(actualName, expectedName);
    }
}
```
